# Sample Projects for Bharat Pi Boards

IoT projects using Bharat Pi for final year students, mini projects, home automation, IoT training, and more.

**Bharat Pi** is an IoT prototyping platform designed for students, innovators, startups, and enterprises. Bharat Pi boards are available in multiple variants to suit your needs across the IoT spectrum of use cases:  

1. **4G/LTE module with ESP32** - Enables rapid prototyping of any IoT use case.  
2. **LoRa module with ESP32** - Ideal for smart metering and remote communication.  
3. **Node Wi-Fi with ESP32** - Perfect for beginners starting with IoT projects.  

### _Launching Soon..._

- **STM32-Based Node with Wi-Fi** (Enterprise Edition)  
- **5G with Wi-Fi and Bluetooth** (Enterprise Edition)  
- **Edge Data Center** (Linux Version)  
- **Matter and Zigbee Modules** (with Wi-Fi and Storage)
